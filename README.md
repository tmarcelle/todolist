# todolist
Single Page to do list application made with html, alpinejs and Claude AI

## Why was this created
Wanted to explore creating a todo list and wanted to have it all in a single HTML Page.

## How to use
Pretty straightforward, download the todolist.html file, open it in a web browser, or add it to your website.
